### 插件介绍
自定义表情包生成器，沙雕图生产效率高
### 插件用法
群内发送：
```yml
# 使用
xxx.jpg
xxx.JPG

# 切换表情包
img 表情包名

# 表情包列表（默认提供一份，后续自定义需自己提供，或取消该功能）
img list
```
自定义表情方法：
1. 查阅插件运行后在 `./plugins/imageCustom/添加表情方法.pdf` 生成的教程
2. 或直接查阅 [在线文档](https://github.com/fz6m/mirai-plugin/tree/master/guide/MiImage)
